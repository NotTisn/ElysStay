- Change healthcheck to a stable endpoint and add startup slack:
  - endpoint: `/realms/master/.well-known/openid-configuration` or `/health/ready` (if enabled)

File: `.github/copilot-instructions.md`
- File is accurate now for docs path, but sits outside both git repos.

Impact:
- Team members do not automatically receive these instructions by cloning backend/frontend only.

Recommendation:

  1. Move a copy into `ElysStay-be/.github/` and track it there, or
  2. Introduce a dedicated workspace repo later (if team agrees).
